# Database Scripts

Scripts SQL para setup e migração do banco de dados Supabase.

## Arquivos

- `supabase_setup.sql` — Estrutura principal das tabelas
- `supabase_fase1_migration.sql` — Migração fase 1 (campos adicionais)
- `supabase_market_alerts.sql` — Tabela de alertas de mercado

## Como usar

Execute os scripts no Supabase SQL Editor na ordem acima.
